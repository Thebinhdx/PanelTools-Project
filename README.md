## ❓How to Use?

### ⬇️Method 1:
- Open **PowerShell**
- **Copy and Paste** the code below, and press **Enter**:
- - #### For Windows 11
  - ```
    irm https://raw.githubusercontent.com/Thebinhdx/PanelTools-Project/refs/heads/main/paneltools.ps1 | iex
    ```
    
<hr>

### ⬇️Method 2:
- Download the script: [PanelTools.cmd](https://github.com/Thebinhdx/PanelTools-Project/releases/latest/download/PanelTools.cmd)
- Run the script.
